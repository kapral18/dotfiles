# GitHub Templates (Agent Guidance)

These files are agent-only guidance for composing PR descriptions and issue bodies.

They are NOT repository `.github/*` templates.
Do not copy them into a repo unless the user explicitly asks.

Pick the closest template for the repo/domain and adapt it.

Files:
- `pr_base.md`: default PR description structure
- `issue_base.md`: default issue body structure
- `pr_elastic.md`: Elastic/Kibana/Elasticsearch-flavored PR description
- `pr_elastic_bugfix.md`: Elastic/Kibana bugfix PR template (includes Before/After)
- `pr_elastic_chore.md`: Elastic/Kibana migration/tech-debt PR template
- `pr_elastic_feature.md`: Elastic/Kibana feature PR template (UI-heavy friendly)
- `issue_elastic.md`: Elastic/Kibana/Elasticsearch-flavored issue body
- `team_kibana_management.md`: Kibana Management-owned areas (CODEOWNERS-derived guidance)

Note: Elastic-flavored templates include an agent-guidance section for common label conventions observed in `elastic/kibana`.
